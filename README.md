# Neural Networks: Zero to Hero — worked by hand

Andrej Karpathy's [course](https://karpathy.ai/zero-to-hero.html) builds neural networks from scratch — starting with a scalar autograd engine and ending with a working GPT. I'm implementing every lesson myself: no AI assistance, no copying from the reference repo.

The rule is deliberate. I want to work in AI engineering, and the fastest way to never understand backprop is to let a model write it for me. Everything here is typed by hand, and `struggles.md` records where I got stuck and why.
